# Phone Profit Calculator

## Overview
This program calculates the profit generated from selling different phone and tablet products during a selected time period. The user can calculate profits for a specific day, the entire week, business days, or weekend days.

## Features
- Calculates profit based on product type and quantity sold
- Supports multiple product categories
- Allows profit calculation for:
  - A specific day
  - The entire week
  - Week business days (Monday–Friday)
  - Weekend days (Saturday–Sunday)
- Displays total profit and a performance message

## Products Included
The program calculates profit for the following items:
- Apple iPhone
- Android Phone
- Apple Tablet
- Android Tablet
- Windows Tablet

Each product has a predefined profit value used to calculate total earnings.

## How It Works
1. The program prompts the user to choose a time range (day, week, business days, or weekend).
2. The user enters a product number and quantity sold.
3. The program calculates the profit for each product.
4. The total profit is displayed along with a message indicating performance.

## Technologies Used
- Python
- Basic data structures such as dictionaries and lists
- Functions for modular program structure
